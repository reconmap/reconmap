The **users** view allows administrators to manage application users, their roles, and access.

![Users view](/images/screenshots/users.png)

From the users list you can:

- Create new users.
- Edit user profiles and roles.
- Enable or disable user accounts.
- View user activity.

Roles include Administrator, Superuser, User, and Client.
