The **project details** view shows all the information related to a single engagement, including summary, description, and client information.

![Project details](/images/screenshots/project-details.png)

From the project details view, you can:

- Edit the project summary and description.
- Manage project membership (who has access to this project).
- View and manage assets associated with the project.
- Report or import vulnerabilities found during the assessment.
- Generate and send penetration test reports.
- Manage vault items (credentials) related to the project.
- View project activity and audit logs.
- Add notes and attachments.

The project summary also shows the risk distribution of vulnerabilities reported for this project.
