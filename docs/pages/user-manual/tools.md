Reconmap includes a set of tools to help pentesters and security engineers in their daily work.

## Vault

The **vault** allows you to securely store credentials (passwords, API keys, etc.) related to your projects.

![Vault](/images/screenshots/vault.png)

From the vault you can:

- Store new credentials.
- Assign credentials to specific projects or users.
- Search for credentials.
- Encrypt and decrypt secrets securely.

## Password Generator

The **password generator** helps you create strong, unique passwords with different configurations.

![Password generator](/images/screenshots/password-generator.png)

You can customize the length and complexity of the generated passwords, including characters, numbers, and symbols.
